# Discord-Poll-Bot

Allows you to create polls for your discord server!

## Syntax

Get help with:
**|poll | help**

Start a new poll with:
**|poll | new | {poll duration in seconds} | {poll question} | {first emoji - answer} ... | {last emoji - answer}**
